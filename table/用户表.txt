用户表 uinfo

	uid 用户的唯一id(主键)
	unick 用户的昵称
	uname 用户账号(全局唯一)
	upwd 用户密码 32的md5值
	usex 用户性别
	uface 用户的头像
	uphone 用户的手机 // 用于手机验证
	umail 用户的邮箱 // 用于游戏验证
	ucity 用户所在城市
	uvip 用户vip等级
	vip_endtime 用户vip结束时间
	is_guest 是否为游客账号
	status 账号状态 // 0 表示正常，1表示封号
	wx_token 微信token //用于微信登录验证
	ugold 用户金币
	uroomCard 用户房卡
	udiamond 用户钻石
